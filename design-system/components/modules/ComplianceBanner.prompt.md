Compliance banner — firm but friendly, informs without nagging, never blocks browsing.

```jsx
<ComplianceBanner tone="warning" title="Not balanced yet">
  You've wagered $23 of $40 across 3 bets. Add $17 and at least 2 more bets.
</ComplianceBanner>
```

Tones: `warning` (incomplete/over), `info`, `success` (balanced). Optional trailing `action`.
