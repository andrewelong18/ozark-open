Bet outcome badge — hit (win/green ✓), miss (loss/red ✕), push (=), void (∅). Color + glyph + label so it survives a long table without becoming a Christmas tree.

```jsx
<OutcomeBadge outcome="hit" />
<OutcomeBadge outcome="void" size="sm" />
```

push/void mean the stake is returned (neutral, quiet).
