Generic pill badge for labels/counts. For bet lifecycle and outcomes, use the semantic `StatusBadge` / `OutcomeBadge` instead so color+label stay consistent across a 25-row table.

```jsx
<Badge tone="indigo">Round 1</Badge>
<Badge tone="gold" uppercase>Featured</Badge>
```

Tones: `neutral | indigo | gold | green | red | amber | solid`. `uppercase` for eyebrow-style labels.
