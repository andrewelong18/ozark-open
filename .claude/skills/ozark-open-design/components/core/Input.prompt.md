Text/email/number input with a warm hairline border and indigo focus ring. 44px tall. Accepts leading/trailing adornments.

```jsx
<Input placeholder="you@example.com" type="email" />
<Input leading="$" type="number" invalid />
```

Set `invalid` for error state (red border). Use `leading`/`trailing` for currency glyphs or icons. For inline whole-dollar bet stakes, prefer `StakeInput`.
